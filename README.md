# LC3-VM

A Implementation of LC3 (Little Computer 3) virtual machine written in Go.
Emulates LC3's insruction set, memory model and execution flow

## Status

Work in Progress

Inspired by Jason Meiners’ [“Write your Own Virtual Machine”](https://www.jmeiners.com/lc3-vm/)
